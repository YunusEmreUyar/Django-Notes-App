# Django Notes App (Keepmo)
[Live Demo](http://keepmo.pythonanywhere.com)

## Installation

    git clone https://github.com/YunusEmreUyar/Django-Notes-App.git    
	cd keepmo
	python -m venv venv
	cd venv/Scripts
	activate
	cd ..
	cd ..
	pip install -r requirements.txt
	cd keepmo
	python manage.py runserver
